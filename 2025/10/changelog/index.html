Diário de atualizações realizadas no Bloco de Notas

Inspirado pelo comentário do @dalbo1201 depois de publicar o post "Um novo começo" no Manual do Usuário, resolvi criar uma página de changelog para acompanhar as atualizações realizadas no Bloco de Notas, sejam de layout ou novas funcionalidades.

A ideia é ir adicionando as atualizações tendo como base a estrutura de "CHANGELOG", no qual cada atualização pode ser considerada um release.

Se tiver interesse, acompanhe as atualizações na página updates.
