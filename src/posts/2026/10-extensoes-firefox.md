Depois de publicar sobre [meus aplicativos essenciais em 2025](https://tcelestino.github.io/bloco-de-notas/2026/01/meus-apps-essenciais-em-2025/), algumas pessoas me perguntaram quais extensões uso no Firefox. Como a grande maioria das pessoas que conheço usa o Google Chrome, resolvi compartilhar as dez extensões que considero essenciais no meu dia a dia.

1. [uBlock Origin](https://addons.mozilla.org/pt-BR/firefox/addon/ublock-origin/): essencial nos dias de hoje com a quantidade de scripts que muitos sites usam para rastrear o usuário. Desabilito para alguns sites.
2. [Bitwarden](https://addons.mozilla.org/pt-BR/firefox/addon/bitwarden-password-manager/): gerenciador de senhas multiplataforma (tem versão para iOS, Android, Windows, Linux, terminal...). Uso a versão gratuita que atende bem para a grande maioria dos usuários.
3. [Enhancer for YouTube](https://addons.mozilla.org/pt-BR/firefox/addon/enhancer-for-youtube/): reduz distrações no YouTube, como comentários, e traz melhorias na interface e no player.
4. [Linguist - web pages translator](https://addons.mozilla.org/pt-BR/firefox/addon/linguist-translator/): uso para traduzir páginas automaticamente. O Firefox tem recurso nativo, mas ele se baseia no idioma do sistema operacional — que no meu caso é inglês. Para traduzir para pt-BR pelo recurso nativo, são necessários vários passos.
5. [Raindrop](https://raindrop.io/): sinto falta da simplicidade do del.icio.us, mas com o Raindrop consigo automatizar utilizando o [IFTTT](https://ifttt.com/). Por exemplo: os vídeos que dou like no YouTube, são adicionados automaticamente no Raindrop.
6. [Notion Web Clipper](https://addons.mozilla.org/pt-BR/firefox/addon/notion-web-clipper/): uso o Notion como um banco de dados, então tenho uma lista de artigos para ler posteriormente, artigos que já li e que posso querer rever, etc.
7. [Marreta](https://addons.mozilla.org/pt-BR/firefox/addon/marreta/): extensão para remover paywall de diversos sites.
8. [Firefox Multi-Account Containers](https://addons.mozilla.org/pt-BR/firefox/addon/multi-account-containers/): uso para isolar os cookies entre serviços que utilizo.
9. [Gesturefy](https://addons.mozilla.org/pt-BR/firefox/addon/gesturefy/): cria ações para o Firefox usando o mouse. Exemplo: mover o mouse para baixo segurando o botão direito fecha a aba ativa.
10. [Redirector](https://addons.mozilla.org/pt-BR/firefox/addon/redirector/): uso para redirecionar automaticamente alguns sites para suas versões alternativas. Por exemplo, todo link do x.com é redirecionado para o nitter.net.
