### Adicionado

- Meta tags padrão no layout base `src/_layouts/default.njk`
  - Meta tag `title` com o título do site
  - Meta tag `description` exibe o `summary` do post quando disponível, ou a descrição padrão do site
  - Meta tag `keywords` usa as `tags` do post quando disponíveis, ou as palavras-chave padrão do site
- Campo `keywords` em `src/_data/site.json` com palavras-chave padrão do blog
- Meta tags Open Graph e Twitter Card no layout base `src/_layouts/default.njk` para compartilhamento nas redes sociais
- Campo `baseUrl` em `src/_data/site.json` com a URL base do domínio para construção de URLs absolutas

- Tag `<title>` atualizada para incluir o `summary` do post como prefixo quando disponível, no formato `"[summary] - [título do site]"`
- Classe `.content__post` adicionada ao elemento `<article>` em `src/_layouts/post.njk` para escopo dos novos estilos de post

- Tamanho de fonte do link "ver todos os posts" alterado de `var(--font-size-small)` para `var(--font-size-base)` em `src/assets/css/last-posts.css`
